# Coincheur confiné

Jouer aux cartes en ligne entre amis sans création de compte

## Setup environment

`docker-compose up` et c'est parti.

Mais pour développer, il est recommandé de faire 

```
docker-compose run --service-ports --rm app bash
$ yarn dev
```
